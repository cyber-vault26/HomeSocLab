# LOGSTASH
This is the termed as the data swiss army knife t acts as a dynamic data processing pipeline 
with a rich plugin ecosystem that allows users to ingest, parse, and transform data in real-time
Common Use Cases:

    Log Processing: Parsing, cleaning, and structuring Nginx/Apache logs.
    Data Enrichment: Adding geographical information to IP addresses.
    Security (SIEM): Ingesting logs into Sentinel or other security systems.
    Database Syncing: Using JDBC to sync data
